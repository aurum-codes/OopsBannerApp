**OopsBannerApp** is a console-based application designed to generate and manage text banners. More importantly, this project serves as a practical implementation of core **Object-Oriented Programming (OOP)** principles, demonstrating how to structure scalable and maintainable code.

## ✨ Features & OOP Concepts Demonstrated

* **Encapsulation:** Banner properties (like text, borders, and dimensions) are safely bundled with the methods that operate on them, preventing unauthorized data modification.
* **Inheritance:** Extensible base classes allow for the creation of various banner styles (e.g., `WarningBanner`, `WelcomeBanner`) without duplicating core logic.
* **Polymorphism:** Dynamic method overriding ensures that different banner types can be rendered using a single, unified interface.
* **Custom Formatting:** Dynamic generation of borders and text alignment based on user input.
